## 功能详情

本项目是使用React实现的一个仿照[SM.MS](https://sm.ms/) 的在线图床。  
核心功能包括：
* 登录注册
* 图片上传 
* 图片定制
* 上传历史展示

## 技术栈

本项目使用到的技术栈包括:
1. create-react-app
2. styled-component
3. React-Router
4. Mobx
5. Antd
6. LeanCloud
7. svg
